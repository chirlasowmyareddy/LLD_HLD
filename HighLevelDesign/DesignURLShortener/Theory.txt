Step 5: How To Generate Short Code

This is where interviewers focus.

Option 1: Random String

Generate:

aB3xP1

Pros:

Simple

Cons:

Collision possible

Need retry logic.

Option 2: Base62 Encoding (Preferred)

Suppose DB ID is:

1001

Convert to Base62.

Characters:

0-9
a-z
A-Z

Result:

g7

Short URL:

short.ly/g7

Pros:

Unique
No collisions

This is usually the preferred answer.

Why Base62?

Answer:

Base62 uses digits, lowercase letters, and uppercase letters, giving 62 possible characters per position. This creates compact URLs and supports a large number of unique combinations.

Why Cache?

Answer:

Redirect operations are much more frequent than URL creation. Caching frequently accessed mappings significantly reduces database load and improves response time.

Why 302 Redirect?

Answer:

302 indicates a temporary redirect. Depending on requirements, we can also use 301 for permanent redirection.
